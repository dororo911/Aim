# Aim Trainer Game

A simple and interactive aim training game built with HTML, CSS, and JavaScript.  
The project helps players practice speed, accuracy, and reaction time by clicking targets as quickly as possible.

## Features

- Interactive target-clicking gameplay
- Score tracking
- Timer system
- Reaction and accuracy practice
- Clean and responsive interface
- Simple game logic with JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Goal

The goal of this project is to create a fun browser-based game that improves mouse accuracy and reaction speed while demonstrating core front-end development skills.
